# 개인 기술 블로그 (https://bumsiku.kr)

Next.js 15, TypeScript, TailwindCSS로 구축된 모던 블로그 웹사이트 프로젝트입니다.

## 주요 기능

- 마크다운 지원 블로그 포스트 관리
- 콘텐츠 관리용 관리자 대시보드
- 반응형 디자인
- SEO 최적화

## 기술 스택

- **프레임워크**: Next.js 15 (App Router)
- **언어**: TypeScript
- **스타일링**: TailwindCSS
- **폰트**: Noto Sans KR (한글/영문 지원)
- **마크다운**: React Markdown
